    class Person

    beschreibt eine Klasse Person, die Briefe von anderen Personen erhalten kann und Briefe an andere Personen senden inkludiert Comparable Das Inkludieren des Moduls 'Comparable' ist insofern sinnvoll, da dieses weitere Vergleichsoperatoren, wie z.B: <,>,= auf Basis des Spaceship_Operators definiert. Somit kann eine eigene Klasse , die Comparable inkludiert,diese auch nutzen.

    definiert den Spaceship-Operator nach alphabetischer Ordnung

    # File person_comparable.rb, line 78
def spaceship_alphabetisch(other)
  return nil unless self.class == other.class
  if (@name <=> other.name) == 0
    if (@strasse <=> other.strasse) == 0
      if (@plz <=> other.plz) == 0
        return @ort <=> other.ort
      end
      return @plz <=> other.plz
    end
    return @strasse <=> other.strasse
  end
  return @name <=> other.name
end
